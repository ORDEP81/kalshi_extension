## Screenshot Guidelines

### Technical Requirements
- **Resolution**: High DPI (2x) for crisp display
- **Format**: PNG for screenshots with text
- **Compression**: Optimize for web without quality loss
- **Consistency**: Use same browser and zoom level for all screenshots

### Content Guidelines
- **Clean Interface**: Hide personal information and irrelevant browser elements
- **Good Examples**: Choose markets with clear, readable odds
- **Annotations**: Add arrows, highlights, or callouts where helpful
- **Context**: Show enough surrounding interface for context

### Capture Process
1. **Setup Extension**: Configure with recommended settings
2. **Navigate to Kalshi**: Use a popular, active market
3. **Capture Screenshots**: Use browser dev tools or screenshot tools
4. **Edit and Annotate**: Add helpful callouts and highlights
5. **Optimize Files**: Compress for web while maintaining quality
6. **Test Display**: Verify screenshots look good in README preview
